# Weather-File-Analyzer-Project
## Overview
The Weather File Analyzer is a beginner-friendly Python program that reads and analyzes weather data from a file. Users can view cities and their average high temperatures, search for specific cities, and add new weather data.

This project is designed to help students practice:
- File handling
- Loops and conditionals
- Basic data processing in Python

## Features
- View all stored weather data
- Search for a city
- Add new city weather data
- Save updates to file
